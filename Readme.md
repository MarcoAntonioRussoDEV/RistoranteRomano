# Ristorante Romano

Concept homepage per un ipotetico ristorante romano con promo San Valentino.

Questo progetto è stato realizzato come parte della "Level Up Challenge" lanciata da Aulab: una sfida creativa per mettere alla prova e migliorare le proprie competenze di sviluppo web front-end.

L'obiettivo era creare una landing page moderna, responsive e accattivante, con animazioni e attenzione all'accessibilità, per promuovere una serata speciale di San Valentino in un ristorante romano.
